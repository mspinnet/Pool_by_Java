Для компиляции проекта используйте: mkdir target ; javac -d target/ src/java/edu/school21/printer/*/*.java

Для запуска используйте: java -cp target/ edu.school21.printer.Program argument1 argument2

Где:
argument1 это два символа, 1 - отображает белые пиксели, 2 - отображает черные пиксели
argument2 это прямой путь до BMP файла
